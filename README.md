Vaksana Dashboard
=================

> Ensure to have NodeJS v6.x MINIMUM.
https://nodejs.org/en/download/package-manager/#debian-and-ubuntu-based-linux-distributions

### Install
```bash
npm install
```

or 

```bash
yarn install
```

### bundle dev mode

```bash
npm run dev
```

### run : flask server


```bash
python test.py
```

> Then go to [localhost:5000](http://localhost:5000)
